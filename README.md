# Ecommerce-Analytics-EndToEnd-decision-making

> From CSV chaos to $9.3M clarity: A complete e-commerce analytics platform built with SQL Server medallion architecture, star schema data modeling, and Power BI. Includes RFM customer segmentation, Pareto analysis, territory performance tracking, and automated returns monitoring.

## 🎯 Business Problem
A multi-region e-commerce business had sales data scattered across CSV files, 
returns logged separately, and zero visibility into profitability by product, 
region, or customer segment.

## ✅ What I Built
- **Data Pipeline**: Medallion architecture (Bronze → Silver → Gold) in SQL Server
- **Data Model**: Star schema with 4 dimensions + 2 fact tables
- **Analytics**: RFM customer segmentation, Pareto analysis, return rate tracking
- **Visualization**: 6 interactive Power BI dashboards

## 📊 Key Business Impact
- $9.32M revenue traceable to product-color level
- Identified 4.63% return rate on specific product (HL Road Tire) for quality review
- Discovered Accessories category = 62.8% margin vs Bikes = 41.6% (revenue driver)

## 🛠 Tech Stack
SQL Server | T-SQL | Power BI | DAX | Star Schema | ETL | RFM Analysis
